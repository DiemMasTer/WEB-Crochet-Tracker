

Welcome to the WEB Crochet Tracker.

This is a simple tool that shouldn't raise safety concerns as it saves progress in YOUR browser

It was written with the aid of AI.

Tested on Brave (Chromium Browser) and Firefox.

# features, overview 
- **track** your projects, **count** your stitches
- write the pattern or paste it
- **save** your data in your browser automatically, **export** it, **import** it
- two themes: "midnight" and "grey scale"
- auto-expand rows
- add **pattern sections**, such as "Head" OR "body" or whatever you want to name them
- add **notes** to your project and rows.
- add **colors** to stitches, or to sections.


# accepted notation

for rounds (also in lower case):

 `Rounds:` 
`Round:` 
`Rnds:`
 `Rnd:` 
`Rows:`
` Row:`
 `R:`

stitch counts: `(30)`, `[30]` 

for stitches (case inventive): 

`sc, inc, dec, flo, blo, ch, slip knot, slst...`

 even full names. 

digits: `6sc`, `sc6`, `6 sc, `sc 6`

repeated rows: `R 1-10:` 

and similar, they get expanded for easier tracking.

repeated stitches groups: `()*5` `()x5` `()X5`, square brackets `[]` also work
groups like these also work `(()*5)*5`

## pattern sections 
you can write stuff like
```
head
R1: ...
R2: ...

body:
R1:...

whatever you want:
R1:...
```
the sections can have they color changed

## notes for rows

`5sc # your note comes after the hashtag`

notes can also be added later from tracker view.

## color 
you can use the color toolbar to add colors, write their tags or use the interface during tracker view. this works for sections, entire rows, groups, stitches inside groups...

```
<white> Head: </white>
R1: (<blu>sc</blu>, inc)*5 
```

## what has to be transformed by user

`sc all around (30)` 

and similar such as `inc all around (30)` has to be manually transformed.

`4 times` or `times 4` has to be manually transformed.

`repeat round 5` has to be manually transformed

this will probably stay this way, maybe I could add a language specific sanitizer.
